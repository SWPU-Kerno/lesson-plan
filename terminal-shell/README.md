## 常见指令
>结合指令演示，顺便介绍一下**相对路径**与**绝对路径**的概念

* ./
* ../../
* /

|      指令      |         作用         |
| :------------: | :------------------: |
|       cd       |       目录切换       |
|      pwd       |     查看当前目录     |
|       ls       | 列出当前目录下的文件 |
|     mkdir      |      创建文件夹      |
|     rmdir      |      移除文件夹      |
|      echo      |       打印内容       |
|     touch      |       创建文件       |
|       rm       | 删除指定文件/文件夹  |
|       cp       |       复制文件       |
|       mv       |       移动文件       |
|      cat       |     查看文件内容     |
|      more      |     查看文件内容     |
|       vi       |       编辑文件       |
| 按键：Ctrl + L |    清空终端中内容    |
| 按键：Ctrl + C |  停止正在进行的任务  |
| 按键：Ctrl + D |       关闭终端       |
